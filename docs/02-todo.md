# 작업 목록

이 파일은 소스에 함께 관리하는 프로젝트 작업 목록이다. 범위, 순서, 설계가 바뀌면 함께 업데이트한다.

## 현재 실행 로드맵

남은 화면은 `전략`, `백테스트`, `검색기` 세 가지다. 구현 우선순위는 다음과 같다.

1. `전략`: 시스템 제공 전략을 늘리고, 선택한 전략으로 종목 후보를 찾는 기능을 먼저 만든다.
2. `백테스트`: 같은 전략 정의를 사용해 과거 성과를 검증한다.
3. `검색기`: 전략과 백테스트가 안정화된 뒤 상세 조건 검색을 실제 데이터에 연결한다.
