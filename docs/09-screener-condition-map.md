# 검색기 조건 맵

이 문서는 검색기 UI 조건, 조건식 표현, 실제 데이터 매핑 상태를 관리한다. 검색기 조건을 추가하거나 전략 데이터 계약이 바뀌면 함께 업데이트한다.

## 기본 원칙

- 조건식은 `AND` 조합을 기본으로 한다.
- 화면의 기본값은 비어 있음 또는 `all`이며, 선택된 조건만 조건식에 포함한다.
- 가격, 거래대금, 수급 금액은 한국 원화 기준으로 표현한다.
- 화면 표시와 입력 기준 시간은 한국시간이다.
- 사용자가 직접 작성한 전략은 `외국인 5일 순매수 >= 500억`, `모멘텀 >= 70`처럼 주요 한글 조건명도 같은 필드로 해석한다.
- 매핑 상태는 `완료`, `부분`, `예정`으로 관리한다.
- 백엔드가 해석하지 못한 조건은 `unsupported_conditions`로 반환하고, 검색기 화면은 이를 `미연동 조건`으로 표시한다.

## 다음 매핑 우선순위

1. 섹터/산업을 KRX 또는 KIS 종목 마스터에서 안정적으로 채운다.
2. 연속 배당 기간은 DB에 저장된 OpenDART 다년 배당금 지급 기록으로 계산한다. 현재 배당수익률, 배당성향, 배당성장률, 연속 배당 기간은 전체 종목 커버리지가 부족하므로 실제 검색 품질이 낮다. 별도 배당 이력 테이블을 만들고 KRX, KIND, KIS, OpenDART 중 전체 종목을 안정적으로 채울 공급원을 붙인 뒤 검색기 배당 조건을 실사용 수준으로 올린다. OpenDART 고유번호 캐시, 단일회사 전체 재무제표 조회, 매출/이익/마진/부채비율/유동비율/ROE/ROA/FCF/EV-EBITDA/배당금 요약 파생, DB 저장, 전략/검색기 후보 보강은 1차 완료했다.
3. 목표가 괴리는 신뢰 가능한 무료 공급원이 확정되기 전까지 핵심 전략 조건에서 제외한다.
4. 조건 빌더는 위 조건식 필드를 그대로 사용해 전략 저장, 전략 후보, 백테스트가 같은 계약을 공유한다.
